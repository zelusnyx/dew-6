New UI for Experiment Design.

For the last part, we are going to ask you to download and try a prototype User Interface for Experiment Design.

The idea before the user interface is to allow users to focus on the behavior and constraints of their experiment, and from this description, create a topology (and generate the NS file) and script for actions and orchestration.
